# **App Features**

- User can sign-up
- User can make Tasks (can have subtasks) and Boards .
- User can track tasks status (Done,Doing ..etc).
- User can drag and drop tasks between boards
- Dark theme

# **Technologies will use**

- Frontend

  - React
  - Tailwind
  - CREATE_REACT_APP
  - _NextJS (maybe)_

- Backend

  - Nodejs
  - Express
  - mongodb
